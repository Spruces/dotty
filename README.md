# Dotty — Simple Dot Painter

Open `index.html` in your browser (no server required) and paint by clicking the grid.

- Left: controls to set grid size, dot size, palette, and brightness.
- Click a color to select it; right-click a palette color to remove it.
- Click grid to paint, right-click (or Shift+click) to erase.
- Buttons: Clear, Export SVG, Export PNG.
